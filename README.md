## Architecture

```
d-ai/
├── core-backend/      # High-performance Go API server
├── crawler-engine/    # Python HTML/CSS crawler & parser
├── ai-assistant/      # Gemini AI integration & chat module
├── frontend-ui/       # React dark-cyberpunk interface
└── system-tools/      # Bash automation & deployment
```

## Quick Start

```bash
bash system-tools/scripts/dev-start.sh
```

| Service | Port | Language |
